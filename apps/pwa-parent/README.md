# PWA Parent

PWA para tutores/padres con primeras pantallas funcionales:

1. `Login`
2. `Resumen`
3. `Niveles`

## Scripts

- `npm run dev --workspace @ecosistema/pwa-parent`
- `npm run build --workspace @ecosistema/pwa-parent`
- `npm run lint --workspace @ecosistema/pwa-parent`

## Variables de entorno

Copiar `.env.example` a `.env` dentro de `apps/pwa-parent` si necesitas personalizar la API.
